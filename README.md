**Stock Data Fetcher**

A simple full-stack application to fetch and display stock market data.
This project includes a React client and a Node.js (NestJS) server.
Please ensure you are using Node.js v22 and pnpm as your package manager.

**Tech Stack **
**Frontend**
React 
React Table
Tailwind 

**Backend**
Node.js v22
NestJS 
Yahoo Finance 

**Tools**
pnpm
Vercel (for deployment)

**SET UP **
npm install -g pnpm

**To Run Client**
cd client 
pnpm i
pnpm run dev


**To Run Server**
cd server 
pnpm i
pnpm run start:dev
